# Virtual Piano 🎹

Bem-vindo ao **Virtual Piano**, um simulador interativo de piano virtual. Este projeto é desenvolvido em **HTML**, **CSS** e **JavaScript** para proporcionar uma experiência musical simples e divertida diretamente no navegador.

---

## 📋 Descrição

O Virtual Piano é um projeto que simula um piano funcional, permitindo que o usuário:
- Toque notas clicando nas teclas do piano com o mouse.
- Use o teclado do computador como controlador para tocar as notas.
- Ajuste o volume do som.
- Alternar a visibilidade das legendas das teclas do piano.

---

## 🚀 Funcionalidades

- **Controle via teclado e mouse**: As teclas do teclado do computador são mapeadas para as teclas do piano.
- **Ajuste de volume**: Use um slider para modificar o volume das notas.
- **Modo alternável**: Oculte ou exiba as legendas das teclas no piano.
- **Animação interativa**: As teclas "piscam" ao serem pressionadas.

## 🖥️ Como Usar

1. Clone o repositório para sua máquina local:
   ```bash
   git clone https://github.com/seu-usuario/virtual-piano.git
   ```

2. Acesse o diretório do projeto:
   ```bash
   cd virtual-piano
   ```

3. Abra o arquivo `index.html` em qualquer navegador moderno.

---

## ⚙️ Controles

- **Teclado**: Utilize as teclas `A`, `W`, `S`, `E`, etc., para tocar as notas correspondentes.
- **Mouse**: Clique diretamente nas teclas do piano.
- **Volume**: Ajuste o controle deslizante para regular o volume.
- **Teclas Visíveis**: Ative/desative a visualização das legendas das teclas.
